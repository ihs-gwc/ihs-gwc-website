IDEs and code editors

Here are some suggestions. There are plenty of other IDEs/code editors to choose from.

IDLE Repl.it Visual Studio Code Atom

[What's an IDE?]

IDE stands for Integrated Development Environment. IDEs are used to edit, debug, and run code. Learn more with this simple explanation .

[How do these IDEs compare?]

IDLE
  • Included with Python when you download Python
  • Easy to use for beginners
  • Doesn't work with other languages
Repl.it
  • Online website (no download required)
  • Easy to share code with others by sending a link
  • Supports many languages
VS Code
  • Download required
  • Offers thousands of extensions
  • Supports many languages
Atom
  • Download required
  • Supports many languages
  • Similar to VS Code
